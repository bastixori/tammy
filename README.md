# tammy

Una hermosa aplicación interactiva y tierna hecha con React. Muestra un corazón central animado, el nombre de Tami y un fondo lleno de maravillas (girasoles) y corazones. Al hacer click en la pantalla, brotan más corazones y girasoles flotantes.

## Contenido
- `index.html`: La aplicación interactiva.
- `background.jpg`: El fondo de pantalla diseñado.
